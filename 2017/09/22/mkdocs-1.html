MKDocs 初探

发表于 2017-09-22

概述

MKDocs是一个项目文档生成器。

可以部署在Github Page供人浏览。

MkDocs可以应用一系列主题，譬如mkdocs,readthedocs,第三方主题，譬如Mkdocs wiki，或者创建自己的主题。

安装

需要环境

更新pip

安装MkDocs

若安装完成，可以用mkdocs --version 查看版本

启用

项目结构

mkdocs.yml      # 配置文件
docs/
    index.md    # 文档首页
    ...         # 其他页面，图片或者其他文件

本地预览

MKDocs自带服务器预览功能，需要使用mkdocs serve指令

打开浏览器，在地址栏输入http://127.0.0.1:8000，即可查看当前项目页面。

基本命令

  • mkdocs new [dir-name] - 创建一个项目
  • mkdocs serve - 启动本地服务器.
  • mkdocs build - 创建文档站点.
  • mkdocs help - 查看帮助.
