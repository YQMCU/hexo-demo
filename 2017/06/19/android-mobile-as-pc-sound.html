用安卓手机作为电脑的播放器

发表于 2017-06-19

话说现在无线耳机、无线音箱什么的甚是流行，不过价格并不便宜。作为喜欢折腾软件的人，如果可以自己免费用软件打造一个套无线音箱/耳机是不是听起来很有趣？！

SoundWire 是一款可以将电脑的声音通过 WIFI 无线网络传送至 Android 手机喇叭上播放出来的有趣软件！如果再在手机上接上音响或耳机的话，瞬间就打造出一款“免费”的无线音箱和无线耳机了。不管是听音乐还是看电影都更加方便了……

无线传输电脑的声音到手机上播放

SoundWire 是一款有创意而又实用的免费虚拟声卡软件，支持将 Windows、Linux 系统以及树莓派的声音通过 WIFI 局域网无线传输串流到你的 Android 手机上播放。看起来好像很高大上，其实 SoundWire 软件的使用还是很简单的，而且也只需要求手机跟电脑是处在同一个局域网里即可。

SoundWire

当然，通过 3G/4G 的移动网络也能实现无线声音串流播放，不过你需要电脑有公网 IP 并懂得如何配置路由器让你的电脑的端口能被公网直接访问，并且还得有足够的手机流量，有具体需求的朋友可以自己研究一下。

SoundWire 使用方法教程

1、首先保证你的手机和电脑是在同一局域网内，然后在 Windows 或 Linux 系统上安装对应 SoundWire Server 服务器端软件并启动。

2、如下图可以看到，SoundWire Server 的界面并不复杂，界面上主要是显示出电脑的 IP 地址，声卡选择、音量大小调节以及将声音录音保存成 MP3 的按钮。这里我们重点是记下电脑的 IP 地址。

SoundWire Server 服务器端

特殊情况：如果你的电脑没有任何声卡，电脑上可能是没有声卡驱动的，这时你可以安装一个 Virtual Audio Cable 的虚拟声卡驱动，否则电脑本身可能无法播放任何声音，自然也就无法传输出去了。此外，XP 系统要求声卡驱动程序支持 Stereo Mix 特性，但迫于某国际唱片组织的压力，目前绝大部分声卡厂商驱动都去除了该功能，其中比较常见的 Realtek 声卡也不支持，所以建议大家还是使用 Windows 8/7 等新系统吧。

3、在 Android 手机或平板设备上安装 SoundWire 的客户端。手机版的 SoundWire 客户端分为免费版以及售价 3.99 美元的完整版。免费版在软件界面上会有广告，并且每隔 30 分钟会有一句很短的女声提示 “SoundWire Free Version”，但实际使用影响并不大。当然，如果此款软件对你很有用，建议购买正版支持一下开发商吧。

4、启动 SoundWire Android 客户端之后，APP 会自动搜索局域网内启动了 SoundWire Server 的电脑。但如果找不到，你只需手工输入电脑的 IP 地址即可连接。一旦连接成功，电脑上的服务器端软件中的 Status (状态) 会变成绿色的 Connected 字样，这时你就能在手机上听到电脑发出的所有声音了。

5、如果你在手机上听到的声音有卡顿现象，首先检查 WIFI 信号是否良好，其次可以设置 SoundWire App 里面的 Buffer 来改善。在 Menu -> Setting 里面有一个叫做 Audio Buffer Size 的设置，将其调小到 8k 可能会对声音卡顿有一定的帮助。

6、最后，是直接在手机的喇叭上播放声音，还是给手机接上音箱或耳机，就视乎你想怎么去利用 SoundWire 了。

特殊情况：如果你的电脑没有任何声卡，电脑上可能是没有声卡驱动的，这时你可以安装一个 Virtual Audio Cable 的虚拟声卡驱动，否则电脑本身可能无法播放任何声音，自然也就无法传输出去了。此外，XP 系统要求声卡驱动程序支持 Stereo Mix 特性，但迫于某国际唱片组织的压力，目前绝大部分声卡厂商驱动都去除了该功能，其中比较常见的 Realtek 声卡也不支持，所以建议大家还是使用 Windows 8/7 等新系统吧.

总结：

虽然 SoundWire 已经是一款比较老的软件了，不过其功能确实很实用。譬如对于电脑没有配备音箱却又需要听到声音的，或者想将电脑的音乐无线串流到客厅音响播放的，又或者想窝在床上看电影而耳机线又不够长的等等各种需求，SoundWire 都能满足你。

如果你也希望能将电脑的声音推送到手机上播出来，或免费打造一个“无线耳机”的话，不妨下载 SoundWire 玩玩吧。当然，你也可以再发挥下想象力看还能利用 SoundWire 做点什么有趣的事情？
