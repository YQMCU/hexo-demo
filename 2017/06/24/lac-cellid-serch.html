基于cellid.cn定位网站的API接口

发表于 2017-06-24

cellid.cn官网[http://www.cellid.cn]

请求说明：

Http请求方式：GET

http://api.cellid.cn/cellid.php?lac=LAC&cell_id=CELLID&coord=COORD&token=TOKEN

token从官网微信公众号获取

参数 必选 类型 说明
lac 是 int 基站lac参数，必须为10进制数
cellid 是 int 基站cellid参数，必须为10进制数
token 是 string 接口使用授权密钥，通过微信获得
coord 否 string 坐标系参数，可选wgs84或gcj02，不提交则默认wgs84

返回说明

参数 说明
lac 基站lac参数
cellid cellid参数
lat 基站坐标:纬度
lon 基站坐标:经度
coord 坐标系:wgs84/gcj02
balance 当日剩余可用接口调用次数

错误的返回

{“Error”:1}

错误代码 说明
1 未传入参数，或传入的参数不完整
2 传入参数格式不正确，必须为整数
3 token密钥错误或无效
4 未查询到结果
5 当日接口调用次数超过允许上限
6 传入坐标参数不正确

调用限制：

免费token：每日调用上限为200次。
